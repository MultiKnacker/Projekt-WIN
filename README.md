## Einleitung
Test
Dieses Repository befasst sich mit der Entwicklung von einem Teil, der Webseite Autovermietung. 
Das Team Beta implementiert folgende features:

- Mietstationverwaltung
- Performance Berichte
- Personalverwaltung
- Fahrzeugverwaltung
